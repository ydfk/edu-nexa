# 当前还缺什么

## 高优先级

- 账号与角色绑定还只是结构占位，还没正式落库
- 教师记录后，监护人查看链路还没有真正闭环
- 学生、监护人、教师三类核心实体还没正式入库
- 作业记录缺少图片、语音、模板短语等真实录入能力
- 小程序微信手机号换取逻辑还未接入真实服务端能力，目前仅提供开发环境手机号回落配置

## 中优先级

- 统计指标口径还没定，例如完成率、异常率、查看率
- 后端接口还未从占位数据切换到真实查询与写入
- 管理后台还缺正式权限校验、筛选、表单、搜索、导出
- 小程序还缺拍照上传、列表筛选、角色切换

## 后续可扩展

- 到校 / 离校签出
- 学生自主查看
- 家长补充反馈
- 财务与课消
